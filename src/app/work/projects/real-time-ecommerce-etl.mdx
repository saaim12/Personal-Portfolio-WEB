## Overview

Built a medallion-architecture (bronze/silver/gold) streaming pipeline moving
Kafka-published orders into a Parquet bronze layer on S3-compatible storage via
PySpark Structured Streaming. Modeled a gold star schema — customer, product, and
date dimensions around a sales fact table — and orchestrated the batch stages with
an Airflow DAG on Docker Compose. Documented the design decisions and trade-offs in
a companion article.

**Tech:** Kafka · PySpark (Structured Streaming) · Airflow · PostgreSQL · Docker Compose · S3
