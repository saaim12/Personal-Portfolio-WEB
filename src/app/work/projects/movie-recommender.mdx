## Overview

Building a production-ready movie recommendation service on MovieLens interaction
data and item metadata. Uses a hybrid approach combining content-based similarity,
item co-occurrence signals, and popularity bias for cold-start stability. FastAPI
endpoints cover health checks, personalized recommendations, similar-item retrieval,
and model reloading, with ingestion, feature engineering, training, evaluation, and
serving separated into layers. Added Docker Compose, optional Redis caching,
API-key-protected endpoints, structured logging, request tracing, and CI-ready tests.

**Tech:** Python · FastAPI · scikit-learn · Airflow · Redis · Docker
