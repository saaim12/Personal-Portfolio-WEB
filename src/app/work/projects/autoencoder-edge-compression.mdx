## Overview

Developed an autoencoder to compress and reconstruct sensor data for lightweight
edge-device communication, tuning latent-vector size for a compact representation.
Converted the encoder for microcontroller-side inference and exposed the decoder
through a Flask API on AWS EC2 with S3 integration for encoded-payload handling.

**Tech:** Python · TensorFlow · TensorFlow Lite Micro · Flask · AWS (EC2, S3)
